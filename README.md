
# 🏥 Medical Data History – SQL Project

## 📌 Overview
This project was developed during an internship program and focuses on analyzing medical datasets using SQL. It demonstrates strong query-writing skills to extract, filter, and summarize patient and admission data.

---

## 🎯 Project Objective
To retrieve, analyze, and summarize medical patient data using SQL in order to answer business and analytical questions.

---

## 📊 Project Description
The project involves writing multiple SQL queries on structured datasets such as **patients** and **admissions**.  
It emphasizes core SQL concepts like filtering, aggregation, grouping, and date-based analysis to generate meaningful insights.

---

## 🛠️ Key SQL Concepts Used
- SELECT, WHERE  
- ORDER BY  
- GROUP BY, HAVING  
- Aggregate functions (COUNT, SUM)  
- Date functions  
- Aliasing  
- Sorting and filtering  

---

## 🚀 Tasks Performed
- Retrieved patient demographic details  
- Filtered patient records based on conditions  
- Analyzed admission trends using date functions  
- Grouped patients based on height and weight ranges  
- Performed aggregation using COUNT and SUM  
- Sorted and formatted query results  

---
